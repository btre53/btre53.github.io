# Requirements: Portfolio Website

**Defined:** 2026-03-14
**Core Value:** Every piece must be immediately understandable by a non-technical recruiter

## v1 Requirements

### Landing Page

- [ ] **LAND-01**: Visitor sees a clean landing page with name, "analyst who builds" positioning, and links to all articles/projects
- [ ] **LAND-02**: Landing page links to LinkedIn and GitHub profiles
- [ ] **LAND-03**: Landing page is mobile-responsive with min 44px tap targets

### Articles - Gambling Industry

- [ ] **GAMB-01**: Article 1 "Pricing & Calibration" — explains how bookmaker odds are prices, implied probability, calibration curves, favourite-longshot bias, and the universal lesson for any pricing business
- [ ] **GAMB-02**: Article 2 "Automated Triage" — explains the compress-rank-summarize pipeline, LLM as constrained junior analyst, directing human attention to highest-impact areas
- [ ] **GAMB-03**: Article 3 "Optimisation & Classification" — explains knapsack-style constrained selection, simple threshold classification using star power, when simplicity beats complexity

### Articles - AIB Scammer

- [ ] **SCAM-01**: Article explains how the phishing site worked (mobile-only, user agent detection, credential harvesting)
- [ ] **SCAM-02**: Article explains the counter-attack (spoofed user agents, fake Irish IPs, 4,000 fake credentials to poison the dataset)

### Live Projects

- [ ] **LIVE-01**: SEC Parser card links to live app with 1-2 sentence description
- [ ] **LIVE-02**: Badminton App card links to live app with 1-2 sentence description
- [ ] **LIVE-03**: Irish Elections card links to existing GitHub Pages article

### Visual Design

- [ ] **STYLE-01**: All articles use Irish elections style (Georgia serif, cream #fafaf8 background, dark green #1b3a2d headers, max-width 720px)
- [ ] **STYLE-02**: Charts are static PNGs with consistent styling (1px border, 4px border-radius)
- [ ] **STYLE-03**: Single responsive media query for screens under 600px

## v2 Requirements

### Additional Content

- **CONT-01**: AlertaAlba explainer article (missing persons data extraction from Twitter)
- **CONT-02**: Place-price article (horse racing pricing model comparison)
- **CONT-03**: Deribit/Polymarket trading system writeup (for quant-specific applications)

### Enhancements

- **ENH-01**: Custom domain (roberttreacy.com or similar)
- **ENH-02**: Dark mode toggle
- **ENH-03**: PDF download option for articles

## Out of Scope

| Feature | Reason |
|---------|--------|
| Blog/CMS | Static HTML only — no dynamic content needed |
| Contact forms | LinkedIn link is sufficient |
| Interactive charts (Plotly/D3) | Static PNGs match existing style, simpler |
| Trading project showcases | Save for quant-specific job applications |
| Unfinished apps (Productivity, QuizUp, Cortix) | Not ready for public display |
